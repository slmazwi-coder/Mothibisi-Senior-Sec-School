# Mothibisi Senior Secondary School Website

## 🌟 Overview
This is the official website for **Mothibisi Senior Secondary School**, a no-fee public school located at Thaba-Chicha A/A, Mt Fletcher, Eastern Cape, South Africa. Our motto is **"We Strive to Excel"**.

## 📊 Key Facts
- **Location:** Thaba-Chicha A/A, Mt Fletcher, 4730
- **Education District:** Alfred Nzo West Education District
- **Municipality:** Elundini Local Municipality
- **Grades:** 8–12
- **Fee Status:** No-Fee Public School
- **Learners:** ±387
- **Educators:** 13
- **Principal:** VS Mgojo
- **Phone:** 084 410 2314

## 🛠️ Tech Stack
- React 19 with TypeScript
- Vite for fast development and builds
- React Router v7 for navigation
- Lucide React for icons

## 🎨 Theme
- **Primary Color:** Maroon (#800020)
- **Secondary Color:** White (#FFFFFF)
- **Accent Color:** Near-Black (#1A1A1A)
- **Font:** Playfair Display (headings), Arial (body)

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ and npm

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## 📁 Project Structure
```
src/
├── components/    # React components (Navbar, Hero, About, etc.)
├── App.tsx        # Main application
├── main.tsx       # Entry point
└── index.css     # Global styles
```

## 🤖 Chatbot
The website includes "Ask Mothibisi" — a rule-based chatbot that answers common questions about:
- Admissions and applications
- School fees
- Academic subjects
- Facilities
- Contact information
